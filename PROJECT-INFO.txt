ğŸ’¡ Example Usage:
==================

1. Import Playlist:
   POST /api/playlists/import
   {

       "http://cdn1.com/stream.m3u8",
       "http://cdn2.com/stream.m3u8"
     ]
   }

3. Access Relay Stream:
   http://localhost:8080/stream/sport

4. Proxy Channel:
   http://localhost:8080/api/proxy/channel/1
